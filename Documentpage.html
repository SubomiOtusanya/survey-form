Object Oriented Programming

Jave is known as an object oriented programming languuage.

This means it is easy to represent entities as objects

an example of this might bea student class to represent a student 

          
            public class Student {
              // Instance variables
              private String name;
              private int age;
              private String studentId;
              
              // Constructor
              public Student(String name, int age, String studentId) {
                  this.name = name;
                  this.age = age;
                  this.studentId = studentId;
              }
              
              // Getter methods
              public String getName() {
                  return name;
              }
            }
          
          
            public class Main {
              public static void main(String[] args) {
                  // Create a Student object
                  Student student1 = new Student("John Doe", 20, "S123456");
                  
                  // Invoke methods on the student object
                  student1.displayInfo();
                  
                  student1.setAge(21);
                  System.out.println("Updated age: " + student1.getAge());
              }
          }
